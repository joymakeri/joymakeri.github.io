Hospital Management System

The Hospital Management System is a full-featured web-based application designed to manage hospital operations such as patient records, doctor schedules, billing, pharmacy inventory, and appointments.

🔧 Technologies Used

  • PHP (Backend)
  • MySQL (Database)
  • HTML & CSS (Frontend)

✅ Key Features

  • Add/View/Edit patient details
  • Appointment booking & management
  • Billing and payment tracking
  • Pharmacy inventory control
  • Secure user login (Admin, Doctor, Staff)
